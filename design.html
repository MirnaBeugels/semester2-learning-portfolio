Portfolio

Toegepast: XD/Photoshop/Illustrator

Voor mijn Portfolio wilde ik graag gebruik maken van SVG animaties. Om die reden ben ik met het ontwerpen van mijn illustratie begonnen. Deze heb ik gemaakt in Illustrator.

Toen ik de basis uitvoering van mijn poppetje had heb ik het ontwerp voor de portfolio website ook in Illustrator gemaakt.

Initieel was het plan om bij iedere maand een bijpassende illustratie te maken. Echter kwam er zo wel heel veel code bij de SVG kijken. Omdat het via GSAP niet mogelijk bleek de SVG niet inline in de HTML te verwerken heb ik besloten het alleen bij de animatie van het gezicht te houden.

Wat ik heb geleerd...

  • Dat je bij het maken van je SVG illustratie de lagen al goed moet benoemen, groeperen en samenvoegen om later de animaties makkelijker te kunnen programmeren. De namen van de lagen worden in de code verwerkt als id namelijk. Deze kun je dan in je JS en CSS aanroepen en er code op toepassen.

Pinkpay

Toegepast: XD/Photoshop/Illustrator

Voor PinkPay heb ik een wireframe gemaakt in XD.

Voor het logo heb ik het originele logo van Pinkpop in SVG formaat opgezocht. De letters 'pop' heb ik vervolgens zelf veranderd naar de letters 'pay' met behulp van het pengereedschap.

Daarna heb ik op basis van het wireframe de afzonderlijke pagina's vormgegeven in Photoshop.

Wat ik heb geleerd...

  • Hoe je met het pengereedschap kunt werken in Illustrator

IXD Wireframe & Header

Toegepast: Figma/Photoshop

Voor het IXD project heb ik een wireframe gemaakt in Figma. De reden dat ik Figma heb gebruikt in plaats van XD is dat mijn medestudenten de voorkeur aan Figma gaven omdat niet iedereen beschikt over het Adobe pakket.

Het ontwerp voor de header heb ik vervolgens gemaakt in Photoshop. Ik vind het zelf het fijnst om alleen de structuur en indeling van een website in een wireframe te verwerken omdat ik in Photoshop meer mogelijkheden heb om het visuele aspect te maken.
